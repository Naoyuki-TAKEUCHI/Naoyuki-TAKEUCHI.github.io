# Naoyuki Takeuchi Research Website v3

Minimal monochrome research website with a restrained Mars-surface accent color.

## Design

- Monochrome base: white / black / gray
- Accent: reddish brown inspired by the Martian surface
- Compact typography: no oversized hero title
- Static HTML/CSS/JS, ready for GitHub Pages

## Publish

Upload the contents of this folder to the root of your `Naoyuki-TAKEUCHI.github.io` repository.
Make sure `index.html` and `assets/` are visible at the top level of the repository.
