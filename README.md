# Naoyuki Takeuchi Research Website

Static personal research website template for GitHub Pages.

## Files

- `index.html` — home page
- `research.html` — research overview
- `publications.html` — papers and presentations
- `talks.html` — talks, outreach, and community activities
- `gallery.html` — research figures
- `cv.html` — CV download page
- `assets/css/style.css` — visual design
- `assets/js/main.js` — theme toggle and BibTeX copy button
- `assets/img/` — placeholder research graphics
- `assets/pdf/` — put your CV PDF here

## Before publishing

1. Replace placeholder profile links in `index.html`.
2. Add your official email address.
3. Add DOI / publisher / PDF links in `publications.html`.
4. Put your CV PDF at `assets/pdf/cv_naoyuki_takeuchi.pdf`.
5. Replace SVG placeholder figures with exported figures from your research.

## GitHub Pages deployment

1. Create a repository named `USERNAME.github.io`.
2. Upload these files to the repository root.
3. In GitHub, open Settings → Pages.
4. Set the source to the main branch and root directory.
5. Visit `https://USERNAME.github.io` after deployment.

## Custom domain

If you buy a domain, add a `CNAME` file containing the domain name and configure DNS according to GitHub Pages documentation.
